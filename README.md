# 🚀 NexusCRM

A modern full-stack Customer Relationship Management (CRM) system built using **Flask**, **SQLite**, **HTML/CSS**, and **SQLAlchemy**.

NexusCRM helps businesses manage customers efficiently through secure authentication, customer tracking, dashboard analytics, and an elegant SaaS-inspired interface.

---

## ✨ Features

### 🔐 Authentication System

* User Registration
* Secure Login & Logout
* Password Hashing using Werkzeug
* Session Management

### 👥 Customer Management

* Add Customer
* View Customers
* Edit Customer
* Delete Customer
* Search Customers

### 📊 Dashboard Analytics

* Total Customers
* Active Customers
* Prospect Customers
* Inactive Customers

### 🏷 Customer Status Tracking

Customers can be categorized as:

* Prospect
* Active
* Inactive

### 🎨 Modern UI/UX

* Glassmorphism Design
* Responsive Layout
* Gradient Effects
* Dark SaaS Theme
* Smooth Animations

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/NexusCRM.git
cd NexusCRM
```

### Create Virtual Environment

```bash
python -m venv venv
```

Activate environment:

**Windows**

```bash
venv\Scripts\activate
```

**Mac/Linux**

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
